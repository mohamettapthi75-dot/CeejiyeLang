# рЯЪА CeejiyeLang v1.0.0 - War-saxaafadeedka Koowaad

Waxaan ku faraxsanahay inaan ku dhawaaqno sii deynta rasmiga ah ee **CeejiyeLang v1.0.0**. Mashruucan waa luuqad programming oo ku dhisantahay Python laakiin ereyadeeda muhiimka ah (keywords) ay yihiin Af-Soomaali.

## рЯМЯ Tilmaamaha Muhiimka ah (Key Features)

- **Transpiler Soomaali ah:** Koodhka ku qoran .cee waxaa loo beddelayaa Python 3 oo sax ah.
- **CLI & REPL:** Waxaad koodhka ka socodsiin kartaa Terminal-ka ama waxaad isticmaali kartaa REPL-ka tooska ah.
- **Error Handling:** Qaladaadka Python waxaa loo turjumay Af-Soomaali si ardayda u fududaato.
- **Testing Framework:** Tijaabooyin diyaarsan oo lagu hubinayo badbaadada luuqadda.
- **CI/CD:** GitHub Actions oo si toos ah u tijaabiya koodh kasta oo cusub.
