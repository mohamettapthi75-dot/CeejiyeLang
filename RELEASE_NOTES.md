# 🚀 CeejiyeLang v3.0.0 - Sii deynta rasmiga ah ee Compiler-ka AST

Waxaan ku faraxsanahay inaan ku dhawaaqno sii deynta rasmiga ah ee **CeejiyeLang v3.0.0**. Kani waa isbeddel weyn oo keenaya "Compiler" cusub oo ku dhisan AST iyo maktabad rasmi ah oo Soomaali ah (Standard Library).

## 🌟 Tilmaamaha Cusub (New Features in v3.0.0)

- **AST Compiler:** Transpiler-ka hadda wuxuu isticmaalaa Tokenization iyo AST, taas oo ka dhigaysa koodhka mid aad u saxan oo aan ku khaldamin strings ama comments.
- **Somali Standard Library:** Waxaan ku darnay `stdlib.py` oo leh shaqooyin Soomaali ah oo rasmi ah sida `tiro()`, `jajab()`, `isku_xidh()`, iyo `kala_jar()`.
- **Error Handling 100% Soomaali:** Dhammaan qaladaadka (Runtime & Syntax) hadda waxaa lagu sharaxay Af-Soomaali oo leh talooyin (tips).
- **Advanced REPL:** REPL-ka hadda wuxuu leeyahay prompts cusub (`ceeji> `), command history, iyo shaqooyin sida `nadiifi` iyo `caawi`.
- **Web Playground:** Hadda waxaad koodhka ku tijaabin kartaa biraawsarkaaga adigoon waxba rakibin.
- **VS Code Extension:** Syntax highlighting iyo snippets loogu talagalay .cee files.

## 📖 Ereyada Muhiimka ah (Keywords & Stdlib)

| Soomaali | Python | Sharaxaad |
| :--- | :--- | :--- |
| `daabac` | `print` | Qoraal soo saar |
| `geli` | `input` | Macluumaad weydii |
| `shaqo` | `def` | Samee function |
| `fasalka` | `class` | Samee class |
| `nafta` | `self` | Tixraaca shayga |
| `isku_day` | `try` | Isku day koodh |
| `khalad` | `except` | Qabo qaladka |
| `isku_xidh` | `''.join` | Isku xidh liis |
| `kala_jar` | `str.split` | Kala jar qoraal |
