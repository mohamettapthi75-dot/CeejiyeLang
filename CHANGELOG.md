# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-02-14
### Added
- Initial release of CeejiyeLang.
- Somali keyword transpiler.
- Somali error messaging system.
- Interactive REPL with multiline support.
- CLI tool with `run` and `repl` commands.
- Standard library with Somali aliases.
- Example programs (calculator, guessing game, chatbot).
- Installation scripts for Linux and Termux.
