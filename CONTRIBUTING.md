# 🤝 Ka-qaybgalka (Contributing to CeejiyeLang)

[English Version Below]

## 🇸🇴 Af-Soomaali

Waan ku faraxsanahay inaad xiisaynayso inaad ka qayb qaadato horumarinta **CeejiyeLang**! Wada-shaqayntu waa midda ka dhigaysa mashruucan mid guulaysta.

### Sida aad uga qayb qaadan karto:
1. **Warbixinta Qaladaadka:** Haddii aad aragto wax aan u shaqaynayn sidii loogu talagalay, fadlan fur "Issue".
2. **Features Cusub:** Haddii aad hayso fikrad wanaagsan, la wadaag bulshada.
3. **Koodh Qoraal:** Waxaad horumarin kartaa lexer-ka, interpreter-ka, ama aad ku dari kartaa ereyo cusub.
4. **Dukumentiyada:** Caawi turjumaadda ama hagaajinta README-ga.

### Habka Shaqada:
- "Fork" garee mashruucan.
- Samee laan cusub (`git checkout -b feature/magacaaga`).
- Samee isbeddeladaada oo ku dar tijaabooyin (tests).
- Soo gudbi "Pull Request" (PR).

---

## 🇺🇸 English

We are excited that you are interested in contributing to **CeejiyeLang**! Collaboration is what makes this project successful.

### How you can contribute:
1. **Reporting Bugs:** If you find something not working as expected, please open an issue.
2. **New Features:** If you have a great idea, share it with the community.
3. **Writing Code:** You can improve the lexer, interpreter, or add new keywords.
4. **Documentation:** Help with translations or improving the README.

### Workflow:
- Fork the repository.
- Create a new branch (`git checkout -b feature/your-name`).
- Make your changes and add tests.
- Submit a Pull Request (PR).
